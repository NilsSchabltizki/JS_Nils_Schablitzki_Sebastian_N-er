Das Projekt soll eine digitale Stempeluhr werden.

Ein Mitarbeiter hat einen NFC Chip an seinem Schlüsselbund.
In der Firma hängt ein Tablet, dass mit einem NFC-Reader verbunden ist.
Der Mitarbeiter hält seinen Chip an den Reader und kann nun auf einen Button klicken und somit angeben, dass er mit seiner Arbeit beginnt.
Genau umgekehrt funktioniert auch die Abmeldung.
Falls der Mitarbeiter seinen NFC-Chip vergessen hat, kann er optional seinen Personalnummer eingeben.

Der NFC-Chip wird von einem Python Programm ausgelesen und per Socket an den JavaScript Server gesendet.
Der Server ist mit der Datenbank verbunden und verwaltet so die gesammte Stempeluhr.

Das Projekt ist an sich fertig, es fehlt lediglich eine View, welche wir noch entwickeln werden.
